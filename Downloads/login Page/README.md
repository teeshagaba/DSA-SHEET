# DSA SHEET

This sheet contains all the important DSA and LeetCode practice problems with concepts. It includes implementations in C++ for various algorithms like prime checking, sieve of Eratosthenes, sorting, searching, and more. Perfect for beginners and intermediates to practice and understand core data structures and algorithms.

## Contents

- Prime number checking and counting
- Sorting algorithms (bubble, insertion, selection)
- Searching algorithms (binary search)
- Array and vector operations
- Mathematical functions and utilities
- And more practice problems

Feel free to contribute or use these for your learning journey!
